# Output Formats V2

Use this file when the user wants a 2.0-style answer with evidence, graph, and ranking.

## 1. Quick Scan

Use for fast lane triage.

Required:

- one-sentence thesis
- stack summary
- primary bottleneck
- strongest evidence
- three next checks

## 2. Evidence Memo

Use for full write-ups.

Required:

- direction
- stack
- evidence ladder
- weakest assumption
- lane score

## 3. Graph Card

Use when the user wants the system shown visually or structurally.

Required:

- end system
- critical components
- company map
- evidence links
- catalyst links

## 4. Catalyst Watch

Use for monitoring.

Required:

- next events
- bullish confirmation
- bearish confirmation
- most sensitive names

## 5. Ranked Lane Table

Use when the user is comparing multiple lanes.

Suggested columns:

- lane
- bottleneck
- evidence level
- consensus level
- catalyst window
- priority
