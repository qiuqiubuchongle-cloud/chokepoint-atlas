# Output Formats

## 1. Directional Lane Memo

Use this first.

Structure:

1. one-sentence thesis
2. what demand wave is driving it
3. stack summary
4. primary bottleneck
5. evidence grade
6. strongest evidence
7. weakest assumption
8. what breaks the thesis
9. what to monitor next

## 2. Watchlist Format

Use only after the thesis exists.

For each name:

- ticker / company
- role in the stack
- why it matters
- what is confirmed
- what still needs validation
- evidence grade
- main risk

Suggested grouping:

- proven executor
- bottleneck pure-play
- second-order beneficiary
- early optionality

## 3. Underwrite Sheet

Use for a deeper follow-up on one company.

Required sections:

- what exact problem it solves
- who depends on it
- what makes it hard to replace
- revenue timing / ramp path
- what evidence is hard vs soft
- evidence scorecard
- what could invalidate the thesis

## 4. Live Market Setup

Use only when current data is available.

Required sections:

- data timestamp
- one-line thesis
- thesis grade and setup grade
- current price context
- starter zone
- add zone
- no-chase zone
- invalidation / stop zone
- first target
- stretch target
- catalysts to watch
- what would make the plan stale

For crypto assets, also include:

- BTC/ETH market regime
- funding / open-interest read if available
- tokenomics or unlock risk
- spot versus perp quality

## 5. Skill Output for Another Agent

If the user asks to "turn this into a skill", output:

- core belief
- step-by-step workflow
- signal dictionary
- anti-patterns
- escalation rules

Keep it procedural. Avoid essay style.
